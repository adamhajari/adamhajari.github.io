magpie

Local-first tool for indexing, analyzing, and pruning multi-TB file collections across external drives. Builds a persistent metadata shadow that stays searchable even after files are deleted or moved to cold storage.

zulip-mcp

An MCP server that gives Claude access to your Zulip channels, so you can ask for summaries of recent activity.

fastai-forum-mcp

An MCP server that gives agents RAG-style access to the fast.ai forum. It searches ~200k community posts with hybrid keyword + semantic retrieval, so agents can answer course questions using up-to-date community knowledge instead of only the 2022 course materials.

calendar snapshot

An Android app that lets you take a photo of a wall calendar, detects events from the image using OCR, and adds them to your Google Calendar.

ok to wake clock

An Android app that shows a large clock and uses green and orange backgrounds to show when it's "OK to wake". This app is meant to help toddlers know when it's time to wake up and when it's time to still be asleep.

spyre

Spyre is an open source python library for generating simple data driven web apps. The code for these apps is minimal and self-contained (no need to write any html, javascript, or css). The github repo includes a number of examples. See some live examples here.
